✅ 何时使用



· 已通过多维度量化选股打分系统确认为持仓候选的标的

· 用户给出单只股票并指定“用交易策略skill分析/生成策略/写交易计划”

· 需要在一个月左右的持仓周期内执行完整的波段管理

· 持仓周期内需明确的建仓节奏、止盈止损、动态保护和盈亏推演



\---



❌ 何时不使用



· 尚未通过选股流程的陌生标的（先用多维度量化选股打分系统）

· 日内超短线或持有超过3个月的中长期投资

· 数只股票同时批量生成对比策略（仅单票独立生成）

· 用户仅要求基本面研究/估值研究等静态分析

2\. 状态诊断与加仓模式自动判定



2A. 五大加仓模式匹配规则（五选一，不可混用）



模式 触发条件（所有条件须同时满足） 四阶仓位结构

A·正金字塔加仓 MACD零轴上穿 + 均线多头排列 + 主力大单持续净流入 + RSI(14) < 75未超买 40% → 30% → 20% → 10%

B·小仓试探+重仓确认 RSI(14) < 50 + 放量下跌 + 布林带下轨附近 + 止跌K线形态未确认 10% → 40% → 30% → 20%

C·等比例分批加仓 MACD金叉零轴附近 + 均线多头未加速 + 主力小幅流入 + RSI 45\~60 25% → 25% → 25% → 25%

D·回调网格加仓 MACD金叉后DIF不破零轴 + 均线多头 + 股价回踩均线支撑 + 缩量止跌 20% → 30% → 30% → 20%

E·突破确认递增加仓 横盘>2周 + 布林带收窄 + 关键压力位明确 + 即将迎来催化 15% → 35% → 35% → 15%



若已有持仓：标的不再匹配标准五模式之一，转为“已有头寸继续管理”模式，新资金只可在浮盈确认或极端折扣价出现时入场。



3\. 信号冲突裁决机制



矛盾组合 场景 裁决规则

组合X 主力持续流入 + RSI > 78 暂停加仓，暂不减仓；观察2日

· 主力方向与RSI方向矛盾时，主力方向优先



4\. 首仓开盘应对规则



4A. 高开四条路径



路径 竞价高开幅度 仓位上限 操作

H1·温和 < 2.3% 首仓上限不变 10:00\~11:00分笔建仓

H2·中等 2.3%\~5.2% 首仓上限不变 等15分钟，量价止跌或分时双底确认

H3·显著 5.2%\~9.5% 首仓上限×50% 等有效回调+缩量企稳后建仓

H4·极端 9.5% 0%，不建仓 全天只看不买



4B. 低开四条路径



路径 竞价低开幅度 仓位上限 操作

L1·温和 < 1.6% 首仓上限不变 竞价1笔+开盘确认后1笔

L2·显著至支撑 1.6%\~3.6% 首仓上限×85% 支撑区分2笔买入

L3·破中期支撑 3.6%\~8% 首仓上限×40% 需特大单流入确认+尾盘买入

L4·破硬止损位 8% 0%，不建仓 全天不参与



4C. 首仓一票否决清单（高低开通用）



否决条件 后果

大盘联动暴跌：沪指同期开盘>2%且无权重护盘 延后至次日

成交量异常放大：首15分钟>前日同期3倍且走低 全天放弃

跌停封单严重：竞价跌停封单>1亿元 首仓推迟



5\. 阶梯仓位生成



加仓2/3除主触发条件外，增加弹性触发与时间熔断规则：



加仓档位 主触发条件 弹性触发（横盘≥5天+MACD放大+主力流入） 横盘熔断

加仓2 放量突破前高压力位 股价在前高下方横盘满5天，允许在前高下方2%区间内提前加仓 横盘满7天仍未触发，放弃加仓2/3，维持当前仓位

加仓3 突破整数关口+收盘站稳2天 若加仓2已按弹性规则执行，加仓3同步下调，满仓上限降至90% 同上，加仓2放弃后加仓3一并放弃



“三三制”横盘熔断计时：



· 第一段3天：正常观察，不操作

· 第二段3天（累计6天）：若未有效突破，且无一日盘中曾触及原触发价≤1%，弹性触发价收紧至99%

· 第三段1天（累计第7天）：触发横盘熔断——放弃本次加仓2/3



熔断后资金处置：剩余现金保留不动，或转移至选股池中下一个信号最好的备选标的。禁止在同一标的上强行补仓。



6\. 盈亏比保护机制

7\. 五阶止盈止损体系



规则层 触发条件 动作 优先级

硬止损 价格 ≤ 硬止损价 全部清仓 最高

Y组合·减仓 主力卖出 + RSI < 50 不建仓不抄底，已持仓减至25% 较高

止盈1·过热 RSI > 78 + 主力单日净流出 > 阈值 动态减仓： 浮盈<8% → 减至60%仓位 浮盈8%\~15% → 减至50%仓位 浮盈>15% → 减至40%仓位 高

止盈3·目标价 （v2.6事件驱动止盈） ① 若标的未来 ≤15个交易日 内存在明确催化事件（如Google I/O、英伟达Rubin发布、财报披露等），止盈3目标价 = min(技术阻力位，事件兑现后的保守定价区间)，策略中须标注“催化事件：XXX，距今X个交易日”。 ② 若无短期明确催化事件，维持技术阻力位优先规则（前高压力区、布林上轨、斐波那契扩展等），机构目标仅作逻辑验证。 分3天执行全清 低

止盈1/2动态减仓原则：浮盈越厚，减仓比例越果断——保护既有利润优先于追逐额外收益。止盈2从止盈1执行后的剩余仓位中计算减仓比例，最终保留仓位随浮盈增加而递减。

· 8B. 首仓阶段（仅执行首仓）

· 8C. 半程阶段（执行至加仓1或中间阶段）：动态最坏情况（移动止损线）+各关键价位盈亏

⚠️ 注意事项



· 数据时效性强制规则

&#x20; · 分析时点为当日收盘后（15:30后）：所有技术指标（均线、RSI、MACD等）必须使用当日收盘价及最新数据计算。

&#x20; · 分析时点为盘中（如午盘）：趋势类指标统一使用上一交易日收盘数据；当日实时价格仅用于判断触发条件，不用于计算均线。

&#x20; · 均线计算必须基于最近N个交易日的实际收盘价精确计算，逐日列出，不得使用“约”或“附近”等估计值。

&#x20; · 违反以上任何一条的数据，策略视为无效，必须重新计算。

· 止盈3目标价强制核实规则

&#x20; · 止盈3目标价必须基于实际K线可验证的技术位置。

&#x20; · 在设定“前历史密集区上沿”前，必须先核查该股的历史最高价、52周最高价，并在策略中注明来源。

&#x20; · 禁止使用未经验证的线性外推值（如“布林上轨延伸至XX元”）作为止盈3的唯一或主要依据。

&#x20; · v2.6新增：若采用事件驱动止盈，必须明确标注催化事件名称、发生日期，并计算“距离催化还有X个交易日”；事件兑现后的保守定价区间需基于历史类似催化剂的平均溢价率或技术阻力位推算，不得臆造。

· 每标的总仓位上限默认100%，但PE>500倍或持续亏损标的可降至50%\~70%硬上限

· 高单价标的（如科创板千元股）可去除仓位硬上限，根据实际200股最低申报进行简化网格

· 硬止损与动态止损可能同时存在于同一持仓的不同层面上——浮盈激活后的移动止损优先触发

· 横盘7天熔断后，剩余资金严禁在同一标的上强行补仓

· 止盈1/2的减仓比例必须在策略输出时根据当前浮盈状态计算，不可机械套用固定值

· 所有建仓动作须避开开盘竞价时段与尾盘异动期（9:30前/14:55后）

· 休市日（法定假日/周末）分析时，必须标注“数据暂停，X月X日开市后所有信号可能发生变化”

· 策略结尾须附最终风险提醒与免责声明
